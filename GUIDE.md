# 블로그 사용 가이드 📖

안녕하세요! 이 문서는 중학생도 쉽게 이해할 수 있도록 블로그 사용법을 설명합니다.

## 🎯 이 블로그가 뭐예요?

여러분이 직접 글을 쓸 수 있는 나만의 웹사이트예요. 
- 📝 마크다운(`.md`)이라는 간단한 파일로 글을 씁니다
- 📅 달력에서 일정을 관리할 수 있어요
- 🗂️ 카테고리별로 글을 정리할 수 있어요

## 📝 글 쓰는 방법 (초보자용)

### 1단계: 어디에 쓸까요?
`blog` 폴더 안에 카테고리별로 폴더가 있어요:
- `에어드랍`: 암호화폐 에어드랍 관련 글
- `바이브코딩`: 코딩 관련 글
- `회화 MASTER/영어`: 영어 회화 공부 글
- `AI 최신 소식`: AI 뉴스

원하는 카테고리 폴더를 선택하세요!

### 2단계: 파일 만들기
**중요!** 파일 이름은 꼭 이렇게 만들어야 해요:
```
년도-월-일-제목.md
```

**예시:**
- ✅ `2025-11-28-영어공부시작.md`
- ✅ `2025-12-01-에어드랍받는법.md`
- ❌ `영어공부.md` (날짜가 없어서 안돼요!)
- ❌ `2025-11-28영어.md` (하이픈 `-`이 빠져서 안돼요!)

### 3단계: 글 내용 작성하기

파일을 열면 맨 위에 이렇게 써야 해요 (이걸 '메타데이터'라고 불러요):

```markdown
---
title: 여기에 글 제목을 쓰세요
tags: [태그1, 태그2, 태그3]
category: 카테고리/서브카테고리
summary: 한 줄로 글 요약
date: 2025-11-28
---

# 본문 시작

여기부터 진짜 글 내용을 쓰시면 됩니다!
```

**실제 예시:**
```markdown
---
title: 영어 회화 첫걸음
tags: [영어, 회화, 초급]
category: 회화 MASTER/영어
summary: 영어 회화를 처음 시작하는 법을 알려드려요
date: 2025-11-28
---

# 영어 회화 첫걸음

안녕하세요! 오늘은 영어로 인사하는 법을 배워볼게요.

## 기본 인사
- Hello! - 안녕하세요
- How are you? - 어떻게 지내요?

매일 10분씩 연습하면 금방 늘어요!
```

### 4단계: 저장하기
파일을 저장하면 끝! 블로그가 자동으로 글을 찾아서 보여줍니다.

## 🖼️ 이미지 넣는 방법

### 1단계: 이미지 파일 준비
`public/images/` 폴더에 사진을 넣으세요.
예: `public/images/영어책.jpg`

### 2단계: 글에 넣기
```markdown
![설명](images/영어책.jpg)
```

**예시:**
```markdown
영어 공부에 좋은 책이에요!

![영어 교재](images/영어책.jpg)
```

## 📚 마크다운 문법 (간단한 꾸미기)

### 제목 만들기
```markdown
# 아주 큰 제목
## 중간 제목
### 작은 제목
```

### 굵게, 기울임
```markdown
**굵은 글씨**
*기울어진 글씨*
```

### 목록 만들기
```markdown
- 항목 1
- 항목 2
- 항목 3

1. 첫 번째
2. 두 번째
3. 세 번째
```

### 링크 넣기
```markdown
[구글로 가기](https://google.com)
```

### 코드 넣기
```markdown
`짧은 코드`

\`\`\`python
print("안녕하세요!")
\`\`\`
```

## 📅 달력 사용하기

메인 페이지에 큰 달력이 있어요. 이렇게 사용하세요:

1. **날짜 클릭**: 그날 일정을 추가할 수 있어요
2. **일정 추가 버튼**: 새 일정을 만들 수 있어요
3. **일정 수정**: 이미 있는 일정을 클릭하면 수정 가능해요

**주의:** 지금은 여러분 컴퓨터에만 저장돼요. 
나중에 구글 캘린더와 연결하면 다른 기기에서도 볼 수 있어요!

## 🗂️ 새 카테고리 만들기

### 방법 1: 폴더 만들기
1. `blog/` 폴더 안에 새 폴더 만들기
2. 예: `blog/운동일기/`

### 방법 2: 중첩 카테고리
1. 카테고리 안에 또 폴더 만들기
2. 예: `blog/운동일기/헬스/`, `blog/운동일기/요가/`

### 방법 3: 사이드바에 표시하기
`src/components/sidebar.tsx` 파일을 열어서 수정하세요.

```typescript
const categories: CategoryItem[] = [
  {
    title: "운동일기",  // 새 카테고리 이름
    href: "/blog/운동일기",
  },
  // 중첩 예시
  {
    title: "운동일기",
    items: [
      { title: "헬스", href: "/blog/운동일기/헬스" },
      { title: "요가", href: "/blog/운동일기/요가" },
    ],
  },
]
```

## ❓ 자주 묻는 질문 (FAQ)

### Q1: 글이 안 보여요!
**해결 방법:**
1. 파일명이 `YYYY-MM-DD-제목.md` 형식인지 확인
2. 파일 맨 위에 메타데이터(`---`로 시작하는 부분)가 있는지 확인
3. 개발 서버를 껐다가 다시 켜보세요 (`Ctrl+C` 누르고 `npm run dev` 다시 실행)

### Q2: 이미지가 안 나와요!
**해결 방법:**
1. 이미지가 `public/images/` 폴더에 있는지 확인
2. 파일 이름을 똑같이 썼는지 확인 (대소문자도 구분해요!)
3. 이미지 경로가 `/images/파일명.jpg`가 아니라 `images/파일명.jpg`인지 확인

### Q3: 한글이 깨져요!
**해결 방법:**
1. 파일을 저장할 때 **UTF-8** 인코딩으로 저장하세요
2. 메모장보다 VSCode 같은 코드 에디터를 사용하세요

### Q4: 카테고리가 안 보여요!
**해결 방법:**
1. `blog/` 폴더 안에 폴더가 있는지 확인
2. `.md` 파일의 `category` 항목이 폴더 이름과 정확히 일치하는지 확인
3. 사이드바에 표시하려면 `sidebar.tsx` 파일에 추가하세요

## 🎨 색깔 바꾸기

블로그 색깔을 바꾸고 싶다면:

1. `src/styles/globals.css` 파일 열기
2. `:root` 부분에서 색상 변경

```css
:root {
  --primary: oklch(0.6723 0.1606 244.9955);  /* 메인 색상 */
  --secondary: oklch(0.1884 0.0128 248.5103); /* 보조 색상 */
}
```

온라인 색상 선택 도구를 사용하면 쉬워요!
